In this program, I practiced JavaScript array methods such as .forEach(), .map(), .filter(), and .reduce(). I used .forEach() to display names with greetings, .map() to transform data like converting Celsius to Fahrenheit and extracting object titles, .filter() to select specific data such as adults and long strings, and .reduce() to perform calculations like total cost, average score, counting occurrences, and flattening nested arrays. I also combined multiple array methods to solve advanced problems like filtering even numbers and squaring them. This helped me understand data manipulation and functional programming concepts in JavaScript.
